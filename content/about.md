## このブログについて

**AIまかせ** は、Claude Code に作らせた個人開発プロジェクトの **現場ログ** を残すブログです。

「**AI で完結しなかった部分**」と「**結果として詰まった技術の罠（日本語情報が見当たらないニッチも含む）**」を中心に書いています。出発点は技術スタックではなく、AIに任せた結果として出てきたものを記録しています。

## 書いている人

副業をしているエンジニアです。普段の開発の多くは Claude Code に任せていて、自分はディレクターっぽい立ち位置でいます。

「**Claude に任せて作らせ、現場で詰まった部分を記録する**」スタイルで書いています。
