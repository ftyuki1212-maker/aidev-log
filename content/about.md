## このブログについて

**aidev_log** は、Claude Code に作らせた個人開発プロジェクトの **現場ログ** を残すブログです。

n8n × Docker × Claude API を中心に組み立てる中で、「**AI で完結しなかった部分**」「**日本語情報が見当たらないニッチ**」「**動いた仕組みの解剖**」を中心に書いています。

## 書いている人

副業エンジニア。Node.js / n8n / Docker は実務、Python は基礎、フロントエンドは初心者。

開発の多くは Claude Code に投げて作ってもらっています。「自分で全部書いた」体ではなく、「**Claude に任せて作らせ、現場で詰まった部分を記録する**」スタイルで書いています。
